# Lung Cancer Data Visualization Dashboard

## Description
This repository contains a web-based data visualization dashboard designed to analyze and explore lung cancer datasets. The project is structured into modular visualization components, tied together by a main landing page. 

Root Directory: Contains the main application entry point (`index.html`, `main.js`), this documentation (`README.md`), and the primary datasets (`lung_cancer.csv`, `lrModelLungCancer.csv`).
Images: Stores static image insights generated from the visualizations (`parallelcoord1Insight.png`, `parallelcoord2Insight.png`, `scatterplotInsight.png`).

## Installation
Because this project relies on vanilla HTML, JavaScript, and local CSV files, there are no complex dependencies or package managers required. The datasets (`lung_cancer.csv` and `lrModelLungCancer.csv`) are already included in the repository root.

However, to properly fetch the local CSV files without encountering browser CORS restrictions, you must serve the files using a local web server. 

## Execution
**Using VSCode Live Server**
1. Open the `finalProject` folder in Visual Studio Code.
2. Locate the `index.html` file in the Explorer pane.
3. Right-click `index.html` and select "Open with Live Server".
4. Your default web browser will automatically open and display the main dashboard.
